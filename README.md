## React Blog
My personal blog application built with React

### Needs
- nvm (to switch to Node.js v14.16.1)
- firebase-tools@9.6.0 (should be installed with Node.js v14.16.1)

## Getting Started
### Install
```bash
nvm use
npm i
npm i -g firebase-tools@9.6.0
```

### Deploy
```bash
sh ./scripts/deploy.sh
```
